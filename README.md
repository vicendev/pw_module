# Module PowerChina - Odoo

Development enviroment for modules in odoo.

## Installation

Use docker-compose for initialize containers.

```bash
docker-compose up -d
```

## Usage

```bash

# display containers
docker ps

# display images
docker images

# display logs containers
docker logs <container_name>
```
